### gitorder
- git init //在当前文件夹下初始化git创建本地git仓库
- git status //查看当前仓库状态，会显示当前添加的文件和修改的内容
- git add //将当前文件添加到暂存区 
- git rm --cached <file> //使用该命令可以将add添加到暂存区的修移除
- git commit //将当前暂存区的修改提交
